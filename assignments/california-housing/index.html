Assignment Overview

You've been hired as a data analyst by a California real estate agency. Your task is to build a predictive model that can estimate house prices based on neighborhood characteristics. This will help the agency price listings accurately, advise buyers on fair market value, and identify which features drive prices.

The California Housing Dataset

This dataset is derived from the 1990 California census and contains information about 20,640 different neighborhoods (census block groups). Each neighborhood is described by several features that help predict median house values.

Feature Description Unit
MedInc Median income in the neighborhood $10,000s
HouseAge Median age of houses Years
AveRooms Average number of rooms per household Count
AveBedrms Average number of bedrooms per household Count
Population Total neighborhood population Count
AveOccup Average household occupancy Count
Latitude Geographic latitude Degrees
Longitude Geographic longitude Degrees
MedianHouseValue Median house value (TARGET) $100,000s

House values are expressed in units of $100,000. So a value of 2.5 means $250,000.
